Channel Discovery

Discover new Discord categories and channels. Existing config and overrides are preserved. New categories inherit the global default retention but are added disabled — enable them manually to include them in cleanup. New channels in already-enabled categories follow the category's default retention.

Scheduled cleanup auto-discovers before each run; manual sync is only needed to enable new categories between runs.
