Global rules

Rules keep or delete specific messages on top of the age-based cleanup. A rule fires when its conditions match a message; keep wins over delete. The most specific level decides: a rule on a channel beats one on its category, which beats a global rule. Most users only need age-based cleanup. Rules are for the cases age alone can't express (for example: keep messages from a moderator, delete bot messages in one category, never touch pinned in channel X).

Channel Discovery

Discover new Discord categories and channels. Existing config and overrides are preserved. New categories inherit the global default retention but are added disabled - enable them manually to include them in cleanup. New channels in already-enabled categories follow the category's default retention.

Scheduled cleanup auto-discovers before each run; manual sync is only needed to enable new categories between runs.
